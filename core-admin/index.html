Introduction

Core Admin is a Appstarter script. It has clean, responsive and user-friendly design. You can create or build a system without worrying about system login/user management, general settings, notications, query builder etc.

Technologies used:
  • PHP CodeIgniter 4 Web Framework
  • HTML 5
  • CSS (Bootstrap)
  • JQUERY
  • AJAX
Key Features:
  • Clean and Responsive Design
  • Built Using Bootstrap
  • Totally Secure System
  • Multilingual System (Database language)
  • RTL Support
  • Layout Options (Light and Dark Mode)
  • Editable & Sortable Lef menu
  • Menu Settings
  • CRUD modeling
  • Installation Wizard
  • Detailed Documentation
  • Runs on PHP 7.4 and Higher Versions

Security:
Varient is a secure system. There are various mechanisms to secure application. These mechanisms are:
  • Cross-Site Request Forgery (CSRF) Prevention
  • Cross-Site Scripting (XSS) Prevention
  • Password Hashing
  • Avoiding SQL Injection

Requirements:
There are some requirements for Varient to work on your server. These requirements are:
  • PHP >= 7.4+
  • cURL PHP Extension
  • Fileinfo PHP Extension
  • Exif PHP Extension
  • Mbstring PHP Extension

Installation

Check your PHP version before start installation. Your PHP version should be 7.4 or a higher version.

License Code

Go to our help desk ( https://reactmore.com/) and create your account on this system. Then generate your license code from the Licenses section (https://reactmore.com/dashboard).
If you want to change your domain name in the future, you can change it with the "Change Domain" option.

After installation, if you see "404 Not Found" error or if you can not login to your site, please check your main directory. There should be a .htaccess file. If you don't have this file, none of your links will work. You can find this file in the main directory of script files. Depending on the operating system you are using, such setting files may be hidden in your computer. In this case, you may not see this file. You can see this file by making the hidden files visible on your computer.
